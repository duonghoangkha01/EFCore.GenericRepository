## [Unreleased]

### Planned for v1.0.0
- Generic repository pattern implementation
- Unit of Work pattern
- Soft delete support with ISoftDeletable interface
- Rich pagination with PagedResult<T>
- Sorting support (OrderBy, ThenBy, OrderByDescending, ThenByDescending)
- Query operations (GetById, GetAll, Find, FindSingle, Any, Count)
- CRUD operations (Add, Update, Delete, Restore, HardDelete)
- Include support for eager loading
- AsNoTracking support for read-only queries
- DbContext extensions for soft delete configuration
- Dependency injection extensions
- Full async/await support
- Synchronous method alternatives
- XML documentation for all public APIs
- Sample projects demonstrating usage
- Comprehensive unit and integration tests
